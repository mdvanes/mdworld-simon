Solo

everything runs pretty fine, but I just found out that wine (specifically World of Warcraft) can't use the sound device at the same time as skype. After some googling I found that in linux, any program gets exclusive access to the soundcard. A workaround can be found in this tutorial. Now, if I start skype with 'aoss skype' and WoW with 'aoss wine Wow.exe -opengl', and configure the sound in winecfg to use the EsounD driver, I can run both at the same time.
